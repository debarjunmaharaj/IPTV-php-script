TV Channel - IPTV News & Video Portal

Thank you for purchasing our script. This documentation will guide you through the installation and management of your new IPTV news portal.

Introduction

The TV Channel PHP Script is a complete, all-in-one solution for creating a dynamic video and news portal. It is designed to be easy to install, manage, and customize. With a powerful admin panel, you have full control over your content, from news posts and categories to site settings and SEO. It is perfect for community news sites, religious organizations, video bloggers, and any IPTV-style website.

Core Features

  • Powerful Admin Panel: Manage your entire website from one easy-to-use interface.
  • Dynamic Post System: Create, edit, and delete news posts with a rich text editor.
  • Category Management: Organize your content into unlimited categories.
  • Dynamic Menu Control: Choose which categories appear in your main navigation menu.
  • Featured Video Section: Highlight a main video on your homepage.
  • Ad Management: Easily place ads in the header and sidebar.
  • Homepage SEO Settings: Control your homepage's meta title, description, and keywords.
  • Custom Permalinks (Slugs): Create user-friendly and SEO-friendly URLs for your posts.
  • Homepage Post Feed & Filtering: Visitors can filter news by category.
  • Pagination: Automatic page numbering for your post feed.
  • Portable URL Structure: Works in root, subdomain, or sub-folder installations.

Server Requirements

Please ensure your server meets the following requirements before attempting to install the script:

  • PHP Version 7.4 or higher
  • MySQL 5.6+ or MariaDB 10.0+
  • Apache Web Server (or Nginx with equivalent rewrite rules)
  • mod_rewrite Apache module must be enabled
  • PDO_MySQL PHP Extension enabled

Installation Guide

Follow these steps carefully to install the script on your server.

  1. Create a Database: Using your hosting control panel (like cPanel), create a new, empty database. Also, create a database user and assign it full permissions to the new database. Note down the database name, username, and password.
  2. Import SQL File: Using a tool like phpMyAdmin, select your new database and import the `database.sql` file provided in the package. This will create all the necessary tables.
    [Screenshot of phpMyAdmin import screen here]
  3. Upload Files: Upload the contents of the script folder (index.php, blog.php, admin.php, .htaccess, config-sample.php, and the Uploads/ folder) to your desired location on the server (e.g., the `public_html` directory).
  4. Set Folder Permissions: This is a critical step. Give the Uploads/ folder write permissions. The most common permission setting is 755. You can do this via your FTP client or file manager.
    [Screenshot of setting folder permissions here]
  5. Configure the Script: Rename the file config-sample.php to config.php.
  6. Edit Configuration File: Open the new config.php file and enter the database details you noted in Step 1.
  7. Installation Complete! Your website should now be live. You can access the admin panel by visiting yourdomain.com/admin.php.

Troubleshooting / FAQ

Problem: When I click a post, I get a '404 Not Found' error.
Solution: This is almost always an issue with .htaccess or your server configuration.
  1. Ensure you have uploaded the .htaccess file to the same directory as index.php. Some FTP clients hide files that start with a dot (.). Make sure "Show Hidden Files" is enabled.
  2. Confirm with your hosting provider that the mod_rewrite module is enabled for your Apache server.

Problem: Image and logo uploads are not working.
Solution: This is a file permissions issue. The web server does not have permission to write files to your Uploads/ directory. Please set the folder permissions for the Uploads/ directory to 755 using your FTP client or cPanel File Manager.
